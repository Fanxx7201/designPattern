## 软件设计七大原则
### 开闭原则
* 定义: 一个软件实体如类、模块和函数应该对外拓展开放, 对修改关闭
* 用抽象构建框架, 用实现拓展细节
* 优点: 提高软件系统的可复用性及可维护性
* 是所有原则的基础
* coding: com.design.principle.openclose
